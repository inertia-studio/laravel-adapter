# Roadmap

## v0.1 — Foundation ✅

> Released 2026-04-05

- [x] Core builder classes (Field, Column, Filter, Action, Layout, Detail)
- [x] Module system with auto-discovery
- [x] Panel system with multi-panel support
- [x] 30 field types
- [x] 8 column types
- [x] 5 filter types
- [x] 6 detail entry types
- [x] 6 action types with confirmation modals
- [x] Form layout (sections, tabs, columns)
- [x] Form validation (auto-generated from schema)
- [x] Form reactivity (reactive fields, dependent dropdowns)
- [x] Simple modules (modal CRUD)
- [x] Relation managers
- [x] List page tabs
- [x] Soft delete support (auto-detected)
- [x] Authorization via Laravel Policies
- [x] Global search
- [x] File uploads
- [x] Authentication (login, guards, middleware)
- [x] Theming (10 presets, dark mode, semantic tokens)
- [x] Navigation groups with badges
- [x] Dashboard pages (custom, auto-discovered)
- [x] 24 PageSchema primitives
- [x] Closure-based form/action handlers on pages
- [x] Widget system (stat, statGroup, layout)
- [x] InlineTable and InlineForm components
- [x] 7 Artisan commands
- [x] Component publishing (3 override layers)
- [x] Toast notifications
- [x] Loading bar
